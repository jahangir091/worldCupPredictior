# World Cup 2026 Predictor

Interactive FIFA World Cup 2026 simulator with live results, group standings, knockout bracket predictions, and a multi-factor match prediction engine.

## Features

- **Dashboard** — Today's results, group leaders, upcoming fixtures (data through June 23, 2026)
- **Groups** — Live standings from completed group matches + simulate remaining fixtures
- **Schedule** — Full 72-match group stage with AI predictions per fixture
- **Bracket** — Round of 32 through Final; pick winners manually or auto-predict
- **Teams** — Browse 48 teams by FIFA rank, form, injuries, and historical WC data
- **Predict** — Head-to-head comparison: historical vs current stats, injuries, tournament form

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:5173
